## TypeScript
- オプショナルプロパティ
- ?をつければ、任意プロパティとして定義できる
type Human = {
  name: string;    // 名前
  age: number;     // 年齢
  hobby?: string;  // 趣味(「?」を付加したことで、オプショナル(任意)なプロパティになる)
}

// 以降のオブジェクトにHuman型を指定する
const member1: Human = {
  name: 'Taro',     // 必須
  age: 24,          // 必須
  hobby: 'running'  // 任意
}
// => OK
```
