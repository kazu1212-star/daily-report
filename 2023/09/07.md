# わかったこと
- リロードされても値を保持するにはローカルストレージを使う
- 関数はオブジェクト。関数名に()をつけることで、関数としてまとめた処理を呼び出すことができる
- 関数式では関数を値として扱っている。 これは、文字列や数値などの変数宣言と同じ定義方法。
```
// 関数式
const 変数名 = function() {
    // 関数を呼び出したときの処理
    // ...
    return 関数の返り値;
};
// Arrow Functionを使った関数定義
const 変数名 = () => {
    // 関数を呼び出したときの処理
    // ...
    return 関数の返す値;
};
```
